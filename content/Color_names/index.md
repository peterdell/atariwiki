## Compute a color value  
### Decimal  
Color = Hue * 16 + Luminance  
### Hexadecimal  
Color = $HL  
### Binary  
||Bit||7||6||5||4||3||2||1||0  
|Color Value |H|H|H|H|L|L|L|L
